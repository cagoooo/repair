# 專案開發進度表 (Project Progress)

> 本文件記錄「智慧校園報修系統」的開發里程碑與完成功能。

## ✅ 已完成階段 (Completed Phases)

### Phase 0.5.0: 管理與通知優化 (Current)
- **發布日期**: 2026/02/18
- **主要功能**:
  - [x] **動態管理員 (Dynamic Admin)**:
    - 實作 Firestore `system/adminConfig` 讀取邏輯。
    - 支援從後台動態新增/移除管理員，無需修改程式碼。
    - 超級管理員 (Super Admin) 權限控管。
  - [x] **通知分流 (Notification Separation)**:
    - 區分「資訊組」與「事務組」維修類別。
    - 實作獨立的 LINE Notify Token 與 Target ID 設定。
    - 支援一鍵同步 Token 功能。
  - [x] **批次刪除 (Batch Delete)**:
    - 管理後台新增選取模式 (Selection Mode)。
    - 支援多選與全選報修單進行批次刪除。
  - [x] **UX 優化**: 
    - 改善 LINE 通知設定的指引與連結。
    - 統一管理介面風格。

### Phase 0.4.0: 離線與同步

  - [x] 背景自動同步 (Auto-Sync)。
  - [x] 離線 UI 狀態提示。
