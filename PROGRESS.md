# 專案開發進度表 (Project Progress)

> 本文件記錄「智慧校園報修系統」的開發里程碑與完成功能。

## ✅ 已完成階段 (Completed Phases)

### Phase 0.7.1: 標籤視覺精點 (Label Refinement)
- **發布日期**: 2026/04/14
- **主要功能**:
  - [x] **地圖標籤簡化**: 移除教室名稱，僅顯示編號，最大程度減少底圖遮擋。
  - [x] **UI 微調**: 縮減標籤內距與文字大小，適配純代碼顯示。

### Phase 0.7.0: 手機版地圖優化 (Mobile UX Upgrade)
- **發布日期**: 2026/04/14
- **主要功能**:
  - [x] **縮放感知標籤系統**:
    - 地圖全覽模式下僅顯示代碼，放大後才顯示完整名稱。
    - 解決手機版標籤過多導致地圖被遮擋的問題。
  - [x] **玻璃擬態標籤設計**:
    - 採用磨砂玻璃質感背景，提升視覺通透度。
    - 針對不同報修狀態提供高對比色。
  - [x] **排版瑕疵修復**:
    - 解決窄長房間文字破碎垂直排列問題，強制水平排列。
  - [x] **選中視覺回饋**:
    - 點擊教室後加入藍色光暈與邊框動畫。
  - [x] **引導 UI**:
    - 新增移動端縮放手勢引導文字。

### Phase 0.6.1: 權限與安全性修復 (Security Patch)
- **發布日期**: 2026/02/25
- **主要功能**:
  - [x] **Firestore 安全規則優化**:
    - 實作資料庫雙重權限驗證，結合前端寫死的超級管理員與 `system/adminConfig` 動態名單。
    - 修復非超級管理員新增或刪除其他管理員時出現的「Missing or insufficient permissions」錯誤。
    - 確保管理員名單內的成員也能夠順利讀寫 `system/adminConfig`，讓事務組長及其他單位擁有真正且完整的後台管理權限。

### Phase 0.6.0: 數據分析與 PWA 優化 (New)
- **發布日期**: 2026/02/19
- **主要功能**:
  - [x] **數據分析儀表板 2.0 (Analytics Dashboard)**:
    - 實作平均維修時間 (MTTR) 統計卡片。
    - 新增 Top 10 熱門報修地點圖表 (Heatmap)。
    - 新增 Top 5 報修王排行榜 (Leaderboard)。
  - [x] **PWA 體驗升級**:
    - 重新設計安裝提示 UI (Glassmorphism)。
    - 修復 Icon 路徑與 Service Worker 快取錯誤。
    - 新增手動安裝觸發按鈕 (於頁尾)。
  - [x] **系統優化**:
    - **頁尾視覺升級**:
      - 移除手動安裝按鈕 (已整合至自動提示)。
      - 導入 Aurora 極光漸層背景與 Glassmorphism 效果。
      - 優化 RWD 手機版排版與間距。

### Phase 0.5.0: 管理與通知優化
- **發布日期**: 2026/02/18
- **主要功能**:
  - [x] **動態管理員 (Dynamic Admin)**:
    - 實作 Firestore `system/adminConfig` 讀取邏輯。
    - 支援從後台動態新增/移除管理員，無需修改程式碼。
    - 超級管理員 (Super Admin) 權限控管。
  - [x] **通知分流 (Notification Separation)**:
    - 區分「資訊組」與「事務組」維修類別。
    - 實作獨立的 LINE Notify Token 與 Target ID 設定。
    - 支援一鍵同步 Token 功能。
  - [x] **批次刪除 (Batch Delete)**:
    - 管理後台新增選取模式 (Selection Mode)。
    - 支援多選與全選報修單進行批次刪除。
  - [x] **UX 優化**: 
    - 改善 LINE 通知設定的指引與連結。
    - 統一管理介面風格。
